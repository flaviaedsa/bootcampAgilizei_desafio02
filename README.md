## Tecnologia

    * Node versão v14.16.1
    * Cypress
    * Chance (Geração de dados aleatórios)
    * Github Actions (CI)
    * Mochawesone (Relatório)

## Instalação & Execução:

    $ git clone <repositório>
    $ cd repositorio/
    $ npm install

## Execução

    $ npx cypress open (abrindo o browser)
    $ npx cypress run (modo headless)
